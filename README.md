# 🔄 Universal Converter

**Universal Converter** — это программа для перевода чисел между различными системами счисления. Поддерживает целые и дробные числа, включая периодические дроби, с произвольной точностью.

## ✨ Возможности

- **Перевод чисел** между системами счисления от 2 до 500
- **Поддержка дробных чисел** (например, `FF.A3` в шестнадцатеричной системе)
- **Поддержка периодических дробей** с автоматическим обнаружением периода (например, `0.(3)` → `1/3`)
- **Работа с большими числами** (BigInteger) — нет ограничений на размер
- **Точный перевод без потери точности** с использованием рациональных дробей
- **Понятный интерфейс** для ввода и отображения результатов

## 🚀 Примеры использования

| Входные данные | Система | Результат (десятичный) |
|----------------|---------|------------------------|
| `1010` | 2 | `10` |
| `FF` | 16 | `255` |
| `10.5` | 10 | `10.5` |
| `0.(3)` | 10 | `1/3` |
| `0.1(6)` | 10 | `1/6` |
| `A.B(3)` | 16 | `10.69921875` |

## 📦 Требования

- **Qt 5** или выше (для графического интерфейса)
- **Компилятор C++17** или выше
- **Git** (для клонирования репозитория)
